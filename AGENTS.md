# AGENTS.md

这个项目是 SuperPnL 的独立设计与实现仓库。

## 协作规则

1. 回答用户时使用中文。
2. 当前阶段优先完善设计文档，不急于实现训练代码。
3. 不要把原始数据、checkpoint、`artifacts/`、`.venv/` 提交进仓库。
4. 任何新增特征都必须说明是否可能产生未来信息泄漏。
5. 评测必须同时包含 no-trade、buy-and-hold、OHLCV-only 和有因子模型，不能只报告有因子的绝对 PnL。
6. 当前阶段不把盘口、成本、流动性作为训练特征；回测如忽略手续费或滑点，必须在报告中明确标注。

## 项目定位

SuperPnL 不以 K 线预测误差作为最终目标。项目长期定位是研究所有直接面向 PnL 的预测模型；当前阶段只针对现货市场，使用历史 OHLCV 和外生因子，预测可交易 edge，并输出 `0..1` 的现货目标仓位。

第一版不做永续、杠杆、借币做空、盘口建模、实盘交易或在线服务。K 线路径预测可以作为辅助任务，但最终评测以 PnL、风险调整收益、回撤和换手为核心。
